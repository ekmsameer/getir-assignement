# Getir Assignement
A simple restful NodeJS CRUD, with expressJS and mongoDB.

## Resources

- Node.Js
- Express.Js
- MongoDb
- Mongoose
- Postman
- BodyParser

## Routes

|          ROUTE            |       HTTP        |      DESCRIPTION      | 
| ------------------------- | ----------------- | --------------------- | 
| /record/                  |       POST        | Get records based on filter | 

## Running

```
npm install
```
npm start
```
npm run dev
```
