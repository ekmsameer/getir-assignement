# Getir Assignement
A simple restful NodeJS CRUD, with expressJS and mongoDB.

## Resources

- Node.Js
- Express.Js
- MongoDb
- Mongoose
- Postman
- BodyParser

## Routes

|          ROUTE            |       HTTP        |      DESCRIPTION      | 
| ------------------------- | ----------------- | --------------------- | 
| /record                   |       POST        | Get records based on filter body params {
	"startDate": "2017-01-26",
	"endDate": "2017-02-02",
	"minCount": 10,
	"maxCount": 3000.98
}| 

## Running

```
npm install
```
npm start
```
npm run dev
```
